How to export from Google Search Console

  1. Go to search.google.com/search-console
  2. Select your property (kloudesim.com)
  3. Click Performance in the left sidebar
  4. Set date range (last 28 days recommended)
  5. Make sure Queries and Pages are toggled on
  6. Click the Export ↓ button → Download CSV
  7. Import that file here

GSC exports contain: Top queries, Top pages, Countries, Devices. This dashboard reads the "Top queries" sheet.
